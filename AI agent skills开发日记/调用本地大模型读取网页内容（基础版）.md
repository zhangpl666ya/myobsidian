# bs4
`beautifulsoup4`是用来修饰网页内容的python包，爬虫神器，与requests一起使用。

```python
import requests
from bs4 import BeautifulSoup

url = '...'
response = requests.get(url)
response.encoding = r.apparent_encoding

soup = BeautifulSoup(response.text,'lxml')
title = soup.find("title")

links = soup.find_all("a")
for link in links:
	print(link.text)
	print(link.get("href"))
```
被soup包装之后，网页内容就会被分解为各种标签分隔的文本，find和find_all函数就是根据标签找内容

# 本地模型调用
发出请求都是用requests
这里是提出请求所以使用requests.post函数
```python
def local_llm(prompt:str):
	response = requests.post(
		"url",
		json={
			"model":"llama3,
			"prompt":prompt
		}
	)
	return response.json()["response"]
```
- url：这里若是ollama的模型统一写为：`http://localhost:11434/api/generate"
- json：是给ollama的基本信息，包括调用哪个模型和什么提示词

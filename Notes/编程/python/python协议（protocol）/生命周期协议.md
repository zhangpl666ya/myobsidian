> **`__init__` 不是构造函数。**
> 
> 真正“创造对象”的，是 `__new__`。

## `A()` 这一行到底发生了什么？

`a = A(1, 2)`

Python 内部流程是：

1. 调用 **`A.__new__`** —— 创建对象（内存级）
    
2. 返回一个实例对象
    
3. 调用 **`A.__init__`** —— 初始化这个对象
    
4. 把结果绑定给 `a`
    

📌 如果第 1 步不返回对象，  
第 2 步根本不会发生。

### 单例模式
```python
class Singleton:
    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance

```

📌 这一步只能在 `__new__` 做。

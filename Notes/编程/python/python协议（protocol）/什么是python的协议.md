我们知道在类中有很多魔法方法，但魔法方法有哪些你知道吗？
>**协议**（protocol） = **一组约定好的魔法方法**
>不需要继承接口
>只要方法名和语义对的上就可以

这是python的***鸭子类型***：
>“*像鸭子一样叫，那就是鸭子*”

但为什么叫做协议而不是C++中的接口呢？
因为C++中是类调用自己的接口
而python中是这个类通过协议可以被函数操作
```cpp
#cpp
A.size()
A.del(x)

#python
len(A)
del A.x
```

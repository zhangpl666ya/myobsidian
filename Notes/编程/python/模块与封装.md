## 加载模块：声明`import`

对于加载内置和第三方模块，Python 提供了该语句。 使用该声明有几种方式，我们这里会简要提及，从最推荐到最不推荐。`import`

### 显式模块导入

显式导入模块会在命名空间中保留模块的内容。 命名空间随后用来指代其内容，中间带“math .”。 例如，这里我们导入内置模并计算π的余弦：
```python
import math
math.cos(math.pi)
#-1
```
### 通过别名导入显式模块

对于较长的模块名称，每次访问某个元素时都使用完整模块名称并不方便。 因此，我们通常会使用“`import ... as ...`”模式来为命名空间创建一个更短的别名。 例如，NumPy（数值 Python）包，作为一个对数据科学非常有用的第三方包，按照惯例，它以以下别名导入：
```python
import numpy as np
np.cos(np.pi)
#-1.0
```

### 模块内容的显式导入

有时你不想导入模块命名空间，而是想导入模块中的几个特定物品。 这可以通过“`from ... import ...`”模式实现。 例如，我们可以仅导入函数和常数：
```python
from math import cos, pi
cos(pi)
```

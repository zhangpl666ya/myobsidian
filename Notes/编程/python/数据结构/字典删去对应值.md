### 方法 1：使用 `del` 语句（最常用）

`del` 可以直接删除指定键对应的键值对，是最基础、最直接的方式。

```python
# 定义一个示例字典
student = {"name": "小明", "age": 18, "gender": "男"}

# 删除键为 "age" 的键值对（即删除值 18）
del student["age"]

print(student)  # 输出: {'name': '小明', 'gender': '男'}

# 注意：如果删除不存在的键，会报错
# del student["score"]  # 执行这行会抛出 KeyError
```

### 方法 2：使用 `pop()` 方法（删除并获取值）

`pop()` 不仅能删除指定键的键值对，还能返回被删除的值，适合需要用到被删除值的场景。

```python
student = {"name": "小明", "age": 18, "gender": "男"}

# 删除键为 "gender" 的键值对，并获取对应的值
deleted_value = student.pop("gender")

print(deleted_value)  # 输出: 男
print(student)        # 输出: {'name': '小明', 'age': 18}

# 可以设置默认值，避免删除不存在的键时报错
score = student.pop("score", None)  # 键不存在时返回 None，不报错
print(score)  # 输出: None
```

### 方法 3：使用 `popitem()` 方法（删除最后插入的键值对）

`popitem()` 会删除并返回字典中**最后插入**的键值对（Python 3.7+ 有序），适合需要逐个删除键值对的场景。


```python
student = {"name": "小明", "age": 18, "gender": "男"}

# 删除最后插入的键值对（这里是 "gender": "男"）
last_item = student.popitem()

print(last_item)  # 输出: ('gender', '男')
print(student)    # 输出: {'name': '小明', 'age': 18}
```

### 方法 4：删除所有值（清空字典）

如果想删除字典中所有的键值对，使用 `clear()` 方法：

```python
student = {"name": "小明", "age": 18, "gender": "男"}

student.clear()  # 清空字典
print(student)   # 输出: {}
```

### 特殊场景：根据值删除对应的键

如果你的需求是 “已知值，删除这个值对应的键值对”，需要先找到值对应的键，再删除：

```python
student = {"name": "小明", "age": 18, "gender": "男"}

# 要删除的值
target_value = 18

# 遍历字典（注意：不能边遍历边删除，所以先找键）
keys_to_delete = [k for k, v in student.items() if v == target_value]

# 删除找到的键对应的键值对
for k in keys_to_delete:
    del student[k]

print(student)  # 输出: {'name': '小明', 'gender': '男'}
```

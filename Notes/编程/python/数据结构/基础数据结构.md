python中基础数据结构有四种：

| 英文名   | 中文名 | 性质    |
| ----- | --- | ----- |
| list  | 列表  | 可变序列  |
| tuple | 元组  | 不可变序列 |
| dict  | 字典  | 可变映射  |
| set   | 集合  | 可变集合  |

# 共同点：
1. **动态类型**：元素均可为不同数据类型
2. **可迭代**：都支持迭代协议，可用for迭代
3. **内存自动管理**：都由python解释器负责内存分配与释放

# 差异：

|特性|列表（list）|元组（tuple）|字典（dict）|集合（set）|
|---|---|---|---|---|
|**类型归属**|可变序列|不可变序列|可变映射（key-value）|可变集合（无键值）|
|**可变性**|支持增删改元素（`append`/`pop`）|不可修改，元素一旦确定无法变更|支持增删改键值对（`del`/`update`）|支持增删元素（`add`/`remove`）|
|**访问方式**|下标索引（`list[i]`）|下标索引（`tuple[i]`）|关键字索引（`dict[key]`）|无索引，需通过`in`判断成员|
|**元素唯一性**|允许重复元素|允许重复元素|**key 唯一**，value 可重复|**元素唯一**，无重复值|
|**有序性**|插入顺序有序（Python3.7+）|插入顺序有序（Python3.7+）|插入顺序有序（Python3.7+，之前无序）|无序（哈希表存储，无固定顺序）|
|**可哈希性**|不可哈希（可变）|可哈希（不可变，元素需均为可哈希）|不可哈希（可变），key 需可哈希|不可哈希（可变），元素需可哈希|
|**底层实现**|动态数组|固定大小的数组|哈希表（开放寻址法解决冲突）|哈希表（同 dict，仅存 key 无 value）|
|**C++ 对应实现**|`std::vector`|`std::tuple`/`std::array`|`std::unordered_map`|`std::unordered_set`|

# 列表、元组、字典、集合的**定义方式差异**

这四种结构的定义语法完全不同，核心是通过符号区分类型，以下是具体对比（含语法、空结构、初始化方式）：

## 一、基础定义语法

|数据结构|符号 / 关键字|基础定义示例|说明|
|---|---|---|---|
|**列表（list）**|方括号 `[]`|`lst = [1, "a", 3.14]`|用逗号分隔元素，方括号包裹|
|**元组（tuple）**|圆括号 `()`（可省略）|`tup = (1, "a", 3.14)`<br><br>`tup_single = (2,)`（单个元素需加逗号）|圆括号包裹，**单个元素必须加逗号**（否则会被识别为普通表达式）|
|**字典（dict）**|大括号 `{}` + `key:value`|`dic = {"name": "Alice", "age": 20}`|用 `key:value` 对表示元素，逗号分隔，大括号包裹|
|**集合（set）**|大括号 `{}` 或 `set()`|`s = {1, 2, 3}`<br><br>`s_empty = set()`（空集合不能用 `{}`）|大括号包裹元素（无键值对），**空集合必须用 `set()`**（`{}` 是空字典）|

## 二、空结构的定义

|数据结构|空结构定义|注意事项|
|---|---|---|
|列表|`lst = []`|直接用空方括号|
|元组|`tup = ()`|空圆括号（单个元素需加逗号：`(x,)`）|
|字典|`dic = {}`|空大括号（默认是字典，不是集合）|
|集合|`s = set()`|必须用 `set()` 函数，不能用 `{}`|

## 三、初始化方式的差异

除了基础语法，还可以通过其他方式初始化，语法也不同：

1. **列表**：
    
    - 推导式：`[x*2 for x in range(3)]` → `[0,2,4]`
    - 类型转换：`list("abc")` → `["a","b","c"]`
    
2. **元组**：
    
    - 推导式（需加 `tuple()`）：`tuple(x*2 for x in range(3))` → `(0,2,4)`
	    - 推导式必须有tuple关键字，不然就成了[[Generator Expressions||生成器]]推导式
    - 类型转换：`tuple("abc")` → `("a","b","c")`
    
3. **字典**：
    
    - 推导式：`{x: x*2 for x in range(3)}` → `{0:0, 1:2, 2:4}`
    - 类型转换：`dict([("a",1), ("b",2)])` → `{"a":1, "b":2}`
    - `dict()` 关键字参数：`dict(name="Alice", age=20)` → `{"name":"Alice", "age":20}`
    
4. **集合**：
    
    - 推导式：`{x*2 for x in range(3)}` → `{0,2,4}`
    - 类型转换：`set("aabbcc")` → `{"a","b","c"}`（自动去重）

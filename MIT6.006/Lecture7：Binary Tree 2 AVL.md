# AVL tree
- AVL tree保持高度平衡，这个性质也叫做AVL性质
	- 若根的左右子树高度差的绝对值不超过一，就称为高度平衡
	- 节点的倾斜度（skew）定义为右子树高度减去左子树高度
	- 树平衡->skew = -1/0/1
